# OpenSpec en este repositorio

- **Metadatos locales:** `.openspec.yaml` en esta carpeta (raíz del árbol OpenSpec).
- **Fuente de verdad:** `openspec/specs/` (dominios: `core`, `public-booking`, `reception`, `inventory`).
- **Cambios activos:** `openspec/changes/` (p. ej. `piloto-reservas-recepcion/`, `public-booking-crea-usuario/`, `mvp-sitio-hotel-web-institucional/` — proposal, design, tasks y deltas donde existan).
- Conceptos: [OpenSpec — Concepts](https://github.com/Fission-AI/OpenSpec/blob/main/docs/concepts.md).
